Web Application Development

All about this course

Web application Development is an Undergraduate course as part of Information System and Technology. The objective of the course is to enable the students to develop web application and publish them using GitHub. The applications used a MVC. The basic pre-requisite of people taking up the course is understanding of HTML, CSS programming. The course also focuses on the ability to program using Python 3.
